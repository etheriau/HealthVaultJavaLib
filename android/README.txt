This sub project provides some minimal support for writing Android applications.  It contains a reusable SDK and a sample application.

To run the Android Sample:

1.  Start Android version of Eclipse
2.  Select File->Import
3.  Choose Android->Existing Android Code Into Workspace
4.  Browse to <hv-download-location>/android
5.  Ensure android-sdk and examples\weighter are checked as Projects to Import
6.  Click Finish
7.  Select Welcome Project
8.  Click Run As Android Application
